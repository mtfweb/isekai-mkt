데이터베이스 설정 (DB 작업)

1. 웹사이트에서 사용할 데이터베이스를 설정합니다. 
2. 현재 작업 중인 PC에서 서버 DB에 접속할 수 있도록 내 PC의 공인 IP를 외부 접근 허용 목록에 등록합니다. 

웹사이트 파일 배포 (FileZilla 작업)

1. FileZilla를 실행합니다. 
2. FTP/SFTP 주소, ID, PW를 입력하여 서버에 연결합니다. 
3. 로컬 웹사이트 소스 코드 내의 DB 접속 설정 파일을 열어 정보를 서버 DB 정보로 수정합니다. 
4. 수정을 마친 웹사이트 파일 전체를 서버의 폴더로 업로드합니다.

연결

1.연결된 도메인 주소로 접속하여 웹사이트가 정상적으로 작동되는지 확인합니다.
